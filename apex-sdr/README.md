# Apex SDR

Apex SDR is the first specialized digital employee built by Atlas Agents Apex.

## Mission

Apex SDR identifies, conversations with, qualifies, follows up with, and schedules meetings with potential clients.

## Value Proposition

Your first digital employee specialized in getting high-quality commercial meetings, available forever.

## Responsibilities

* Receive leads.
* Start conversations.
* Ask qualification questions.
* Detect commercial intent.
* Follow up with prospects.
* Schedule meetings.
* Deliver qualified opportunities to the human team.

## Initial Metrics

* Leads contacted.
* Leads qualified.
* Meetings booked.
* Average response time.
* Follow-ups completed.
