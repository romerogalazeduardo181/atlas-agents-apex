# Apex Core

Apex Core is the shared foundation used by all Apex Digital Employees.

## Modules

### Brain

Decision-making logic of the agent.

### Knowledge

Company memory, documents, FAQs, and business context.

### Conversations

Message history, conversation state, dialogue flow, and communication channels.

### Actions

Tasks executed by the agent, such as follow-ups, scheduling, notifications, and CRM updates.

### Dashboard

Visual layer for clients to track results, usage, and performance.

## Rule

Any Apex Core component should be reusable by at least three future digital employees:

* Apex SDR
* Apex Legal
* Apex Support
